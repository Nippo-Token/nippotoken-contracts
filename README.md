# NippoToken

## Requrements
- truffle v5.1.43 (Solidity v0.5.16 (solc-js), Node v14.8.0, Web3.js v1.2.1)
- ganache test rpc.

## Getting started
```
$ npm install 
```

## Deploy NIPPO token
```
$ export SEED=<your seed phrase> 
$ truffle migrate --network bsc_testnet
```

## Test
```
$ truffle test
```
